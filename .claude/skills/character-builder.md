# 角色构建技能

## 技能描述

帮助作家创建立体、生动的角色档案，通过一系列引导性问题深入挖掘角色的方方面面。

## 使用场景

- 创建新角色
- 完善现有角色
- 解决角色扁平化问题

## 工作流程

1. **基础信息收集**
   - 姓名、年龄、性别
   - 外貌特征
   - 职业和社会地位

2. **性格塑造**
   - 核心性格特质（3-5个）
   - 优点和缺点
   - 习惯和癖好
   - 说话方式和行为模式

3. **背景故事**
   - 成长环境
   - 重要经历
   - 创伤和转折点
   - 家庭关系

4. **动机与目标**
   - 核心欲望
   - 表层目标vs深层需求
   - 恐惧和弱点
   - 价值观和信念

5. **关系网络**
   - 与其他角色的关系
   - 盟友和敌人
   - 导师和门徒

6. **成长弧线**
   - 起点状态
   - 转变过程
   - 终点状态
   - 学到的教训

## 提示示例

"让我们来创建一个新角色。首先，这个角色在故事中扮演什么角色？（主角/配角/反派）"

"告诉我这个角色的名字和基本信息。"

"这个角色最突出的性格特质是什么？"

"什么经历塑造了这个角色的性格？"

"这个角色在故事中想要达成什么目标？"
