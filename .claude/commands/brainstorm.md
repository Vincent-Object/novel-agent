# 头脑风暴命令

进入创意激发模式，帮助用户进行小说创意头脑风暴。

## 执行内容

当用户使用 `/brainstorm` 命令时，开始一个结构化的头脑风暴会话：

1. **确定方向**
   - 询问用户想要探索的方面（题材、情节、角色、设定等）

2. **发散思维**
   - 提供多个创意方向
   - 使用"如果...会怎样？"的问题
   - 结合不同元素产生新想法

3. **收敛整理**
   - 总结生成的创意
   - 帮助用户筛选最有潜力的想法
   - 建议下一步行动

4. **记录要点**
   - 将关键创意整理成清单
   - 标注优先级

## 示例对话

```
Agent> 欢迎来到创意头脑风暴！让我们一起探索你的小说创意。

你想要探索哪个方面？
1. 故事主题和核心冲突
2. 独特的情节点子
3. 有趣的角色设定
4. 世界观和背景设定
5. 其他（请说明）
```
