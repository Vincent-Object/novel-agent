# Novel Agent - 小说创作智能助手

## 项目概述

Novel Agent 是一个基于 Claude Agent SDK 构建的智能小说创作助手，旨在帮助作家进行小说构思、大纲规划、角色设计、情节发展和内容撰写。

## 核心功能

### 1. 小说构思 (Novel Ideation)
- 协助生成小说创意点子
- 探索主题和题材
- 提供不同类型小说的创作方向建议

### 2. 大纲规划 (Outline Planning)
- 创建详细的章节大纲
- 规划故事结构（三幕式、英雄之旅等）
- 管理情节线索和时间线

### 3. 角色发展 (Character Development)
- 创建丰富的角色档案
- 设计角色关系网络
- 追踪角色成长弧线

### 4. 内容撰写 (Content Writing)
- 协助撰写场景和对话
- 提供写作风格建议
- 保持叙事一致性

### 5. 世界观构建 (World Building)
- 设计虚构世界的设定
- 管理世界观细节
- 确保设定的一致性

## Agent 行为指南

作为一个小说创作助手，你应该：

1. **创意支持**: 在用户需要灵感时提供创意建议，但尊重用户的创作方向
2. **结构化思考**: 帮助用户将模糊的想法转化为结构化的小说元素
3. **一致性维护**: 追踪已创建的角色、情节和设定，确保故事的连贯性
4. **风格适应**: 理解并适应用户的写作风格和偏好
5. **反馈循环**: 及时总结和确认，确保理解用户意图

## 工作流程示例

### 新建小说项目
1. 收集基本信息（类型、主题、目标读者）
2. 创建项目文件夹
3. 初始化角色、大纲和设定文件

### 角色创建
1. 收集角色基本信息（姓名、年龄、背景）
2. 深入挖掘性格、动机和冲突
3. 保存到角色档案

### 撰写章节
1. 回顾大纲和之前的内容
2. 协助撰写新场景
3. 确保与整体故事保持一致

## 技术约束

- 使用 TypeScript 和 Claude Agent SDK
- 所有小说数据以 JSON 或 Markdown 格式存储
- 支持多个并行的小说项目
- 实现良好的错误处理和用户反馈
